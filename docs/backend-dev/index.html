In this page you'll find some important piece of information to help you understand the backend development applied to this project.

HTTP codes#

Throughout the code, you may see a lot of numbers like: 200, 404 or 403. These are HTTP codes that indicate the state of a request if it was successful or not. You can learn more about this here: HTTP Status Codes from REST API Tutorial website
